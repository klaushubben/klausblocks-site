Perlin noise becomes the instrument: familiar, expressive, and endlessly playable. Small shifts in scale, drift, color, and decomposition pull the image into motion, while layered passes create rhythm, syncopation, and the feeling of a loop changing as it repeats. Generated entirely on-chain, each iteration is a self-contained visual phrase: fixed in code, but always moving.

After the deterministic image resolves, live block data is polled through public RPC endpoints and grouped into epochs, letting the surface continue to shift in time with the chain. If live data is unavailable, the token hash provides a deterministic fallback.
